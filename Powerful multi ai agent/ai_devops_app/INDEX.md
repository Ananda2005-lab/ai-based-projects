## 🚀 Quick Commands

### Windows
```powershell
cd ai_devops_app
.\setup.bat
.\venv\Scripts\Activate.ps1
python app.py
```

### macOS/Linux
```bash
cd ai_devops_app
chmod +x setup.sh
./setup.sh
source venv/bin/activate
python app.py
```

Then open: **http://localhost:5000**
